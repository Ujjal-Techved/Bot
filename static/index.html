üìã Usage Examples

Basic Usage:

// Include the script <script src="chatbot-widget.js"></script> // Initialize <script> const chatbot = new IRISChatbot(); </script>

With Custom Options:

const chatbot = new IRISChatbot({ position: 'bottom-left', // 'bottom-right', 'bottom-left', 'inline' containerId: 'my-container', theme: { primaryColor: '#ea580c', secondaryColor: '#dc2626', textColor: '#374151', backgroundColor: '#ffffff' } });

API Methods:

// Open the widget chatbot.open(); // Close the widget chatbot.close(); // Send a custom message chatbot.sendCustomMessage('Hello!'); // Destroy the widget chatbot.destroy();

Auto-initialization with HTML attributes:

<!-- Add this div anywhere in your HTML --> <div data-iris-chatbot data-position="bottom-right" data-container="chatbot-container"></div> <!-- Widget will auto-initialize when page loads -->
